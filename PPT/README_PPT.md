# TensorSwitch Presentation

Dynamic presentation showcasing the TensorSwitch journey.

## Quick Start

**Open the presentation:**
```bash
firefox Diyi_tensorswitch_presentation.html
```
Or double-click the HTML file.

No installation needed - everything is self-contained.

## Navigation

- **Arrow keys**: Navigate slides (←/→ for timeline, ↓ to dive deep, ↑ to return)
- **Space**: Next slide
- **P**: Open presenter view with speaker notes

## Presenter Mode

Press **P** to open speaker console in a new window.

Shows:
- Timer
- Speaker notes
- Current and next slides

Navigate in the main window - console updates automatically.

## Presentation Structure

6 phases:
1. **Origin** - Where TensorSwitch came from
2. **Development** - Technical foundation
3. **Personal Tool** - CLI workflow
4. **GUI** - Empowering scientists
5. **AI Power** - Intelligent assistance
6. **Validation & Future** - Current status and plans
